# 财联社快讯爬虫使用说明

本爬虫用于自动抓取财联社（https://www.cls.cn/telegraph）的实时快讯，并按照日期和小时进行本地存储。

## 功能特点
- **定时抓取**：每隔 1 分钟自动请求网页，检查是否有新快讯。
- **去重逻辑**：通过新闻唯一 ID 确保不会重复抓取相同内容。
- **本地存储**：
  - 每天产生一个以日期命名的文件夹（例如 `2026-02-12`）。
  - 每小时产生一个新的 JSON 文件（例如 `10.json`）。
  - 存储的是当天**累计**的新闻（即 `23.json` 会包含当天 0 点到 23 点的所有新闻）。
- **异常处理**：具备基本的错误重试和网络异常处理能力。

## 环境要求
- Python 3.x
- `requests` 库

安装依赖：
```bash
pip install requests
```

## 运行方法
在终端中执行以下命令启动爬虫：
```bash
python cls_spider.py
```

## 文件结构示例
```text
.
├── cls_spider.py
├── 2026-02-11/
│   ├── 00.json
│   ├── 01.json
│   └── ...
└── 2026-02-12/
    ├── 00.json
    ├── 01.json
    └── ...
```

## 注意事项
- 爬虫运行过程中请保持网络连接。
- 如果需要手动停止，请按 `Ctrl+C`，程序会自动保存最后一次抓取的数据后退出。
